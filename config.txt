ERP é uma sigla derivada do nome Enterprise Resource Planning que significa "Planejamento dos recursos da empresa". As principais funções fornecidas são Controle de Estoque, Gerador de Sped, Emissão de Nota Fiscal, Pedido de Compras e Vendas, Lançamento de Cotação, Relatório de Gestão, Cadastro de Produtos, Fornecedores, Cliente e Funcionários e Cotação e Sugestão de Compras Automatizadas. A IntelliCash fornece uma Gestão Completa e Eficiente para a Saúde Financeira da Empresa do Cliente, com a integração de todas as informações, gerando confiabilidade e rastreabilidade de todo o capital. Uma ferramenta foi desenvolvida para simplificar o processo de recebimento e empréstimo de vasilhames. Ela é acessada no menu Gerenciamento→Estoque→Controle de Vasilhames. O estabelecimento precisa de um terminal na recepção, com uma impressora para emitir tickets. Antes de usar a ferramenta, é crucial configurar os produtos, indicando quais são vasilhames e definindo seus preços. Após o cadastro dos vasilhames, é preciso associá-los aos produtos correspondentes. Isso é feito na seção "Outras Informações" do cadastro do produto. Para visualizar vasilhames e produtos associados, vá para Gerenciamento→Estoque→Controle de Vasilhames→Vasilhames. No Painel de Consultas, há duas abas: "Localizar" e "Entrega". A primeira é usada para encontrar entregas ou empréstimos, com opções de filtro. Na aba "Entrega", é possível cadastrar e gerenciar as entregas de vasilhames. Na aba "Empréstimo", são tratados os vasilhames emprestados aos clientes no passado, com opções para finalizar, receber, vender vasilhames, renovar empréstimo e estornar recebimento. Em caso de divergência entre a quantidade entregue e emprestada, novos empréstimos ou tickets de entrega podem ser gerados para ajustar as quantidades. O estorno de recebimento é possível apenas para empréstimos já recebidos e exclui notas ou renovações de empréstimos existentes.

A ferramenta Importar NF Laticínio é um utilitário desenvolvido para facilitar a importação de planilhas de leite para a geração de notas fiscais. A planilha deve seguir um formato específico, contendo informações como data de emissão, CPF do produtor rural, código de barras do produto, quantidade de leite, valores, entre outros. Ao executar o programa, o usuário pode buscar o arquivo da planilha e verificar as informações contidas nela. Após verificar e confirmar a precisão dos dados, o usuário pode clicar no botão "Importar p/Intellicash". Este botão realiza a importação em três etapas: armazenamento em uma tabela temporária, validação cadastral (produtos, produtores rurais, transportadoras) e, finalmente, a importação das notas fiscais para o Intellicash. Se algum registro na planilha não estiver correto ou houver problemas durante a validação cadastral, a operação será abortada para garantir a integridade dos dados.

Para instalar o aplicativo da Intellicash, acesse o link http://intelliware.com.br/download/intellicash e faça o download do instalador. Ao executar o instalador, clique em “Avançar” e certifique-se de fechar todos os aplicativos abertos. Você será apresentado com dois modos de instalação: Server e Client. No modo Server, prepare o servidor configurando o Banco de Dados e instalando os serviços de conexão. Se desejar, instale o Sincronizador ScannTech e o GNFe. No modo Client, prepare o terminal para se comunicar com o servidor. Durante a instalação, forneça informações como diretórios de instalação, endereço IP do servidor e nome do Banco de Dados. Após configurar as opções desejadas, conclua a instalação e escolha se deseja criar atalhos na Área de Trabalho e executar o Intellicash imediatamente após a conclusão. Ao finalizar a instalação, clique em “Concluir” para finalizar o processo.

É crucial manter o sistema Intellicash sempre atualizado para aproveitar as constantes melhorias implementadas. Antes de atualizar, verifique a versão do servidor no EnterpriseServer.exe e confira no Manual de Atualização qual a versão recomendada para o IC. Certifique-se de que o arquivo EnterpriseServer.ini esteja configurado corretamente para a atualização das máquinas Clients. Na máquina Client, tenha o IntellistoreUpdater.exe junto ao Intellicash3.exe para atualizar o aplicativo principal. O IntellistoreUpdater.exe é configurado pelo arquivo Intellistore3.ini na pasta de sistema. Para atualizar o IC, substitua o Intellicash3.exe no servidor e execute-o. O IC perguntará se deseja atualizar, e ao confirmar, selecione o arquivo .GSS para atualização do banco de dados. Faça uma cópia de segurança do banco de dados e escolha o local para salvá-la. Após a cópia, o processo de atualização do banco de dados iniciará automaticamente. Não interrompa o processo para evitar corromper o banco de dados. Após a atualização bem-sucedida, verifique se o IC não está preso nos processos do Windows. Em caso de erro, salve o log e envie para a IWS, feche o IC e restaure o banco de dados. O banco de dados de backup estará em modo de Shutdown; para reativá-lo, use o comando gfix -online -user [usuário] -pass [senha] [banco de dados], apenas se necessário.
